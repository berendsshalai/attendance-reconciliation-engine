LinkedIn

I built an open-source Attendance Reconciliation Engine as part of the Sha-Lai Berends Systems Lab. It compares fictional planned shifts with fictional clock events and turns differences into an auditable exception queue.

The point is not to claim payroll compliance. The point is to show how manual spreadsheet checking can become a local, testable, privacy-conscious system with clear review evidence.
